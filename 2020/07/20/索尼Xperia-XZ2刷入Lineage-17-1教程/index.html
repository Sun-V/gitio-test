索尼Xperia XZ2刷入Lineage 17.1教程
发布于：2020-07-20 10:25:10 标签：/ Android / 访问：

因为此ROM已经获得了Lineage官方的支持，直接刷入official-build版可以在线OTA更新，所以原来的教程就没有太大意义了。下面介绍直接刷入official-build版的步骤。

直接刷入official-build

0.1. 确保手机已经升级到了官方Android 10并且至少启动一次，确保手机已经解锁；

0.2. 通过此链接下载platform-tools R28并解压，进入platform-tools文件夹，此目录下打开命令窗口；

0.3. 通过此链接下载vbmeta.img，在此页面下载Lineage Recovery和lineage-17.1包，通过此链接下载copy-partitions.zip，都放入platform-tools文件夹内；

0.4. 音量增加键进入蓝灯模式（或者adb reboot bootloader进入），在命令窗口输入

0.5. 去除数据线，音量降低键+电源键进入rec，使用音量键选择，并用电源键确定选择Apply update - Apply from ADB，连接数据线，使用adb sideload copy-partitions.zip 刷入，刷入时会有报错，直接点击yes即可；

0.6. 回到主界面，选择并确定Factory Reset, Format data / factory reset 格式化数据；

0.7. 回到主界面，选择并确定Apply update - Apply from ADB，连接数据线，使用adb sideload lineage-xxxxx.zip 刷入LineageOS包；

0.8. 若不需要GApps，root权限等，可直接开机；

0.9. 若需要刷入其他包，选择Advanced - Reboot to Recovery，按0.7. 步骤刷入即可（注：GApps需要在第一次开机之前刷入）；

0.10. 需要索尼相机，需要在这里下载，刷入SemcCamera-xxx.zip和Magisk-xxx.zip后，在Magisk中安装selinux_permissive_v2.0_evdenis.zip 。

建议进行以下操作前对手机内重要数据进行备份，退出SIM卡和消除Google帐号。若执行完下述操作出现卡SONY或无限重启等问题，建议按照步骤仔细地重新刷入。以下操作中，使用第三方固件为Sjll的基于官方的LineageOS 17.1，与之对应，使用底包为52.1.A.2.1（提取码：l98l）。

本文工作主要是对已有的操作步骤进行翻译和说明等，并非具有原创性，读者可进入以下直接或间接使用的项目进行支持：

1. 解锁

关于解锁操作，可以参考这篇教程。

2. 强刷底包固件

强刷操作同样可参考1中给出的链接。刷入结束后，断开数据线，启动系统至桌面。

3. 刷入VoLTE

若不需要VoLTE，此步骤可跳过。

关机后通过音量降低键进入绿灯模式，在此链接中下载oem.zip，解压后进入oem文件夹，打开newflash.exe，输入两次n并回车，出现“请按任意键继续”说明刷入完成。开机，使得状态栏出现VoLTE的标识。

4. 刷入LineageOS

4.1 在此界面下载[Beta]-lineage-17.1-akari-2020-5-26-sjll.zip；通过此链接下载platform-tools R28并解压。将[Beta]-lineage-17.1-akari-2020-5-26-sjll.zip解压后的所有文件放入到platform-tools文件夹中。

4.2 关机后通过音量增加键进入蓝灯模式，进入platform-tools文件夹，在文件资源管理器地址栏输入cmd回车进入命令行窗口，依次输入并回车下面命令

5. 刷入更新包

5.1 在此页面下载更新包，并将压缩包放入platform-tools文件夹中。

5.2 断开数据线，按住音量降低键和电源键开机进入lineage recovery，使用音量键和电源键选择“Apply update - Apply from ADB”，连接数据线，使用adb sideload lineage-xxxxx.zip 刷入更新包。

若不需要GApps，索尼相机，root权限，可以跳过下面操作。

6. 其他

6.1 可在此界面下载Magisk-v20.4.zip，在此界面依ARM64->Android 10.0->Pico, Nano or Micro下载Gapps，在此界面下载索尼相机。

6.2 将所需刷入的压缩包放入platform-tools文件夹中，和刷入更新包的方式一样，使用adb sideload命令刷入。注意，Gapps需要在刷入LineageOS第一次开机之前刷入，索尼相机和Magisk刷入更新包后需要启动一次后再刷入；刷入时会有报错，直接点击yes即可；开机之后的安全中心警报可划走忽略。
